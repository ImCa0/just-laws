一个简洁且便捷的中华人民共和国法律文库

立即访问：[justlaw.cn](https://www.justlaws.cn)

## 特点

- 简洁至上

  以文档的风格展示每一部法律，结构清晰、界面整洁，带来优雅、纯净、极致的阅读体验。

- 访问便捷

  一个网址，适配 PC 和移动端，随时随地触手可及，自带搜索功能，使法律条文的查询更加简单、快速。

- 动态更新

  基于 Claude Code 自动化收录流程，实现法律文档的高效收录和更新。当已收录法律完成修正或修订时，将及时进行更新，保证内容的时效性。
